# jQuery (jquery)
jQuery is a fast, small, and feature-rich JavaScript library that makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

**URL:** [Visit APIs.json URL](https://jquery.com)
